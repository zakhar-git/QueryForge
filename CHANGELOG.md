# Changelog

## 0.4.0

- Removed the version line below the ASCII banner.
- Added Yandex to the recommended source set for the main entity types.
- Added Yandex-specific templates using quotation, site, host, URL, language and MIME operators.
- Expanded the catalog from 67 to 125 sources and from 435 to 711 templates.
- Expanded web, news, social, code, archive, company, publication, technical, map and blockchain sources.
- Added additional CIS company registries for Armenia, Kyrgyzstan, Georgia and Moldova.
- Added source and template coverage tests for the expanded catalog.

## 0.3.0

- Replaced TXT export with a self-contained HTML report.
- Added source, text, depth and category filters to the report.
- Added grouped results, copy actions, theme switching, print layout and responsive styles.
- Changed the default report destination to `output` in the current working directory.
- Added `QUERYFORGE_OUTPUT` for custom export paths.
- Expanded the catalog from 49 to 67 sources.
- Expanded the template pack from 301 to 435 queries.
- Added CIS-focused search, social, map, company-registry and procurement sources.
- Added HTML escaping, offline-resource checks and exporter tests.
- Updated package data so the HTML template is included in installed builds.

## 0.2.0

- Replaced arrow-key navigation with numeric menus and Enter.
- Removed framed panels and promotional interface text.
- Added persistent Russian and English interface modes.
- Added configurable accent, page size, depth, source mode, history and screen clearing.
- Added automatic input type detection.
- Added 19 data types, 49 sources and 301 query templates.
- Added source groups and manual multi-selection with ranges.
- Added browser opening, local history and reruns.
- Added TXT, JSON, CSV and Markdown exports.
- Added YAML-driven source and entity packs.
- Added automated catalog, rendering, detection, CLI and repository tests.

## 0.1.0

- Initial interactive prototype.
