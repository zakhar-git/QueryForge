# QueryForge

QueryForge создаёт поисковые запросы и прямые ссылки для открытых источников. Для запуска не нужны аргументы командной строки.

```bash
python main.py
```

## Возможности

- Управление цифрами и Enter без навигации стрелками
- Русский и английский интерфейс
- 19 типов данных
- 125 открытых источников
- 711 поисковых шаблонов
- Базовая, расширенная и полная глубина
- Рекомендуемые, групповые, все или выбранные вручную источники
- Поиск Yandex: веб, картинки, карты, новости и видео
- Автоматическое определение типа значения
- Локальная история
- Открытие ссылок в браузере
- Автономный HTML-отчёт, JSON, CSV и Markdown
- Расширение через YAML
- Настройки и история в `~/.queryforge`
- Отчёты в `output` или в каталоге из `QUERYFORGE_OUTPUT`

## Типы данных

Ник, email, человек, телефон, Telegram, социальный профиль, компания, бренд, домен, URL, IP, ASN, GitHub-репозиторий, хеш, криптоадрес, документ, изображение, место и фраза.

## Установка

```bash
git clone <repository-url>
cd QueryForge
python -m venv .venv
```

Windows:

```bash
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

Linux и macOS:

```bash
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

## Проверка

```bash
pip install -r requirements-dev.txt
pytest
```

## Примечания

QueryForge не собирает аккаунты автоматически и не утверждает, что найденные совпадения относятся к одному человеку или одной организации. Внешние сервисы могут требовать вход, показывать CAPTCHA, ограничивать доступ по региону или менять публичные адреса поиска.
